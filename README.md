# growi-plugin-codeblock-extended

GROWI ページ内の Markdown コードブロックを拡張する Script プラグイン。

## 機能

### 実装済み

- **ファイル名ラベル** — ` ```言語:ファイル名 ` 形式で書いた場合のみ、コードブロック左上にタブ風でファイル名を表示する。ラベル底部はコードブロックの下レイヤーに潜り込む形で密着し、ラベルテキストは選択・コピー可能。GROWI デフォルトのファイル名表示は非表示にして置き換える。ファイル名を指定しない場合は何も表示しない。
- **言語アイコン** — ファイル名ラベル左端に言語アイコンを表示する。主要言語は専用カラーアイコン、未登録言語は汎用ファイルアイコン（白半透明）。`{lang=xxx}` オプションでアイコン言語を上書き指定可能。`data-no-lang` で非表示。
- **行番号** — コードブロック左側に薄いグレーで行番号を表示する（Zenn 風）。横スクロール時も行番号は左に固定。コピーボタンでコピーされるテキストには行番号は含まれない。
- **開始行番号指定** — コードフェンスに `{start=N}` を付けると行番号を N から開始する。ファイルの一部を抜粋するときに実ファイルの行番号を表示できる。
- **行ハイライト** — コードフェンスに `{hl=行番号[,範囲...]}` を付けると指定行の背景色を強調表示する。行番号は表示される番号（`start` オフセット適用後）で指定する。
- **差分表示** — diff コードブロックの追加・削除・変更なし行を背景色で強調し、左ガターに `+` / `−` 記号を表示する。追加行・コンテキスト行には行番号を付与（`{start=N}` で開始番号指定可能）。`language-diff` と `{diff}` 修飾子の 2 種類の記法をサポート（詳細は「構文オプション」参照）。
- **コピーボタン** — コードブロックにマウスオーバーすると右上にコピーボタンが表示される。
  - 通常コードブロック: クリックでコード全体をクリップボードにコピー
  - diff ブロック: 通常クリックで新コードのみをコピー（`+`/空白行の先頭記号を除去、削除行・ハンク行は除外）。Shift+クリックで raw diff テキストをコピー
  - ホバー時に tooltip が表示される。diff ブロックでは Shift+クリックのヒントを含む
  - コピー成功時: 2 秒間 ✓ アイコン（緑）でフィードバック
  - diff raw コピー成功時: 2 秒間 ± アイコン（青）でフィードバック
  - コピー失敗時: 2 秒間 ✕ アイコン（赤）でフィードバック
- **印刷最適化** — 印刷時はツールバー・行番号・diff ガターを非表示にし、白背景・黒文字でコードを出力する。長い行は折り返して紙面内に収める。ファイル名ラベルは印刷時も表示される。

## 動作要件

- GROWI v7 以降（Script プラグイン対応）
- HTTPS 環境（`navigator.clipboard` API の利用に必要）

## インストール

GROWI 管理画面 `/admin/plugins` でこのリポジトリの URL を指定してインストール。

## opt-out 属性

機能ごとに独立して制御できる。

| 属性 | 効果 |
|------|------|
| `data-no-copy` | コピーボタンを非表示 |
| `data-no-filename` | ファイル名ラベルを非表示（GROWI デフォルトも非表示のまま） |
| `data-no-lang` | 言語アイコンを非表示 |
| `data-no-linenum` | 行番号・ハイライトを非表示 |
| `data-no-diff` | 差分ガター・背景色を非表示 |

## 構文オプション

コードフェンスの言語指定の後ろに `{key=value}` を付加する。

| 構文例 | 効果 |
|--------|------|
| ` ```python:{start=10} ` | 行番号を 10 から開始 |
| ` ```python:{hl=3,5-7} ` | 3 行目と 5〜7 行目をハイライト |
| ` ```python:{start=10,hl=12,14-16} ` | 開始 10・表示番号 12, 14〜16 行をハイライト |
| ` ```python:file.py{start=10,hl=12} ` | ファイル名・開始番号・ハイライトを併用 |
| ` ```none:config.env{lang=bash} ` | `none` 言語でも Bash アイコンを表示 |
| ` ```diff ` | diff 形式の差分表示（Prism の diff 文字色のみ、シンタックスハイライトなし） |
| ` ```python:{diff} ` | Python シンタックスハイライト + 差分背景色・ガター |
| ` ```python:src/app.py{diff} ` | ファイル名ラベル + シンタックスハイライト + 差分表示 |

`hl` の数値は**表示される行番号**を指定する（`start` オフセット後の値）。

### オプション指定の記法ルール

**ルール 1: ファイル名がなくても `:` コロンが必要**

GROWI はコードフェンスの `:` 区切りを `言語:ファイル名` として処理し、ファイル名部分（`{...}` を含む）を `<cite>` 要素に書き出す。このプラグインは `<cite>` からオプションを読み取るため、コロンなしでは `{...}` が DOM に現れず、オプションを検出できない。

```
✓ python:{start=10}        # コロンあり・ファイル名なし → 動作する
✓ python:file.py{start=10} # コロンあり・ファイル名あり → 動作する
✗ python{start=10}         # コロンなし → 動作しない
```

**ルール 2: オプションは 1 つの `{...}` にまとめる**

複数の `{...}` ブロックを並べた場合、先頭の 1 つしか読み取られない。

```
✓ python:{start=10,hl=3,diff}   # 1 つにまとめる → 全オプション有効
✗ python:{start=10}{hl=3}       # 2 つに分けると {start=10} のみ有効
✗ python:{start=10}{hl=3}{diff} # 同上、{start=10} のみ有効
```

```html
<pre data-no-copy><code>...</code></pre>
<pre data-no-filename><code>...</code></pre>
```

## 開発

```bash
pnpm install
pnpm build      # dist/ を更新
```

ビルド後は GROWI 管理画面でプラグインを削除 → 再インストールして反映する。
